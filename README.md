# fivetwooh-site

Public company website for FiveTwoOh, Inc.

Static HTML/CSS. No build step, no JavaScript, no tracking.

## Structure

- `index.html` — home
- `about.html` — about page
- `contact.html` — contact page
- `privacy.html` — website privacy statement
- `styles.css` — shared styles

## Local preview

Open `index.html` directly in a browser, or serve the directory with any static file server:

```
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

This repository is configured to deploy to Vercel as a static site. No build command is required.
